# Marketing Ops (Sekersoft Lojistik)

Bu klasör, **Sekersoft Lojistik** için kampanya kurgusu + UTM standardı + ölçümleme event sözlüğü gibi “operasyon dokümanlarını” içerir.

## Canlıya çıkmadan önce

- **GTM/GA4/Meta Pixel**: `VITE_GTM_ID` (önerilen) veya fallback olarak `VITE_GA4_ID` / `VITE_META_PIXEL_ID` değerlerini ortam değişkenlerine ekleyin.
- **Consent**: Sitede çerez paneli vardır; analitik/pazarlama script’leri yalnızca kullanıcı onayı sonrası yüklenir.
